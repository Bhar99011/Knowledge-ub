# knowledgehub- Library Management System with python, Streamlit & Gemini AI Chatbot  

This project is a **Library Management System** built using **Streamlit** and integrated with **Google's Gemini AI** for chatbot functionality.  

##  Installation & Setup  

Follow these steps to set up and run the project:  

### Install Dependencies  
```bash
pip install streamlit sqlite3 bcrypt base64 google-generativeai
pip install tabulate

### Get Gemini AI API Key
Visit Google AI Console and create an API key.

### Create a Virtual Environment
python -m venv venv

### Activate the Virtual Environment
.\venv\Scripts\Activate

### Run the Application
streamlit run main.py
